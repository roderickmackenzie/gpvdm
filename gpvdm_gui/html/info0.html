If gpvdm can't do exactly what you want it to do, then I am in general happy to add code to make it solve your problem, ideally with the view to some type of joint publication.


Why gpvdm?

In short: The world is facing an existential CO2/energy crisis, as well as a growing population which needs medical care. Thin film devices such as OPVs/OLEDs/OFETs offer many answers to these problems. I feel the more people who are studding these devices the better a chance we have of finding answers.


The roadmap

  • Improved perovskite models, which include ion diffusion.
  • Improved OLED models: I have not used the OLED model my self, so I think there is a lot which could be added to it. Including far/near field calculations.
  • The inclusion of thermal models. There is a thermal model in the model but I have not really used it much.
  • Integrated FDTD optical model.
  • Better fitting with experimental data.
  • Improved OFET models. As with the OLED model, I have not used the OFET model my self so I expect there are a lot of features which could usefully be added to it.
  • Better tools for adding materials to the database.
  • I would be nice to finish off the 3D device model, so irregular shaped contacts could be added.
  • Simplified 3D diode models so that panels can be modelled.
  • Any other features you can imagine......
